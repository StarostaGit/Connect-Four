Here are the semantics for writing the code in this repo (Connect-Four)

 > Class names use normal UpperCamelCase, e.g. class MyClass
 
 > Function names use camelCase, e.g. def sizeOfCircle()
 
 > Variable names use camelCase, e.g. widthOfWindow
 
 > Constant names use UPPERCASE, e.g. NUMBER_OF_ROWS
 
 > File names are created as follows: typeName.extension, e.g. classBoard.py
	EXCEPTION: main.py and all the namespaces
